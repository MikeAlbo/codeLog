Form Group
with NG-Model

Created: May 19, 2016

A basic example of a form input. This example takes advantage of Bootstrap's styling using "form-group" and "form-control". The example also uses Angular's "ng-model" to create a user model within the scope.

Framework Directive Use
Bootstrap form-group associates the elements with the group
Bootstrap form-control controls the styling of the input section
Angular JS ng-model creates/ applies changes to the model within the scope
